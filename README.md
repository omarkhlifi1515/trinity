# Trinity - Repository Overview

This repository contains multiple components of the Trinity Ecosystem (Web, Phone, Agent variants).

Canonical identity & behavior specification: `TRINITY_IDENTITY.md` (root of repository).

Overview of folders:
- `blue trinity/` - Agent scripts and monitoring tools. See its `README.md` for details.
- `green trinity/` - Web application (HR portal, tasks, templates).
- `green trinity local/` - Local development variant of the green web app.
- `red trinity/` - Android & Parrot backend tooling for penetration testing (separate concerns).

This repository has been aligned to the Trinity AI OS behavior and documentation; for policies, capabilities, and operational scenarios consult `TRINITY_IDENTITY.md`.
